# codex-security-test

A simple Python data processing utility for testing purposes.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from main import process_data, calculate_average

result = process_data(["a", "b", "a"])
avg = calculate_average([1, 2, 3])
```
